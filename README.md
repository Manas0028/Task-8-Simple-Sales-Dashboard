# 📊 Task 8 – Simple Sales Dashboard Design

## 📌 Project Overview

This project demonstrates the creation of an interactive **Sales Performance Dashboard** using **Microsoft Power BI**. The dashboard provides valuable business insights by visualizing sales trends, state-wise performance, category-wise sales distribution, and key performance indicators (KPIs). It is designed to help business users monitor sales performance and make data-driven decisions.

---

## 🎯 Objective

The objective of this project is to build a clean and interactive dashboard that:

* Tracks monthly sales performance
* Compares sales across different states
* Analyzes sales by product category
* Displays important KPIs
* Enables interactive filtering using slicers

---

## 🛠️ Tools & Technologies

* Microsoft Power BI
* Microsoft Excel / CSV Dataset
* Data Visualization
* Business Intelligence

---

## 📂 Dataset

**Dataset Used:** `Sales_store_full_data.csv`

Key Columns:

* Order Date
* State
* Category
* Amount (Sales)
* Profit
* Quantity

---

## 📈 Dashboard Features

* 📅 Monthly Sales Trend (Line Chart)
* 📍 Sales by State (Bar Chart)
* 🥧 Sales by Category (Donut Chart)
* 📊 KPI Cards

  * Total Sales
  * Total Profit
  * Total Orders
* 🎛️ Interactive State Slicer

---

## 📊 Key Business Insights

* Monthly sales trends help identify periods of high and low business performance.
* State-wise analysis highlights the highest revenue-generating locations.
* Category-wise analysis reveals which product categories contribute the most to total sales.
* Interactive slicers enable dynamic filtering for faster business analysis.

## 🚀 How to Use

1. Clone this repository.
2. Open `Task8_Sales_Dashboard.pbix` in Microsoft Power BI Desktop.
3. Refresh the data if required.
4. Use the **State Slicer** to filter dashboard visuals interactively.
5. Explore sales trends and business insights.
